# Tred — социальная сеть для тредов

**Tred** — это новая социальная сеть для обсуждений и тредов.

Платформа доступна по адресу:

https://tred.su

Tred сочетает:

- формат обсуждений и тредов
- бесконечную ленту
- ветвящиеся комментарии
- стены (аналог сообществ)

Проект ориентирован на русскоязычный интернет и находится на ранней стадии развития.
