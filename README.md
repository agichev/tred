# Tred

**Tred** — это новая социальная сеть для обсуждений и тредов.

Платформа сочетает привычный интерфейс современных социальных сетей с форматом обсуждений, где один пост может развиваться в длинный тред с комментариями.

🌐 Сайт проекта:  
https://tred.su

---

## О проекте

Tred — это социальная платформа, ориентированная на обсуждения, треды и сообщества.

Основная идея проекта — создать удобную площадку для общения и обсуждений, которая будет сочетать:

- формат тредов
- современный интерфейс социальной сети
- простую и понятную структуру

В отличие от классических форумов или Reddit-подобных платформ, Tred делает упор на **простоту интерфейса и удобство использования**.

---

## Основные возможности

На данный момент в Tred доступны:

**Треды**  
Посты, вокруг которых строятся обсуждения.

**Ветвящиеся комментарии**  
Ответы могут развиваться в полноценные дискуссии.

**Стены**  
Аналог сообществ или тематических пространств.

**Бесконечная лента**  
Контент отображается в привычной для социальных сетей форме.

**Система уведомлений**  
Лайки, подписки и системные события собираются в одном месте.

**Пуши**  
Быстрые сообщения через систему уведомлений.
